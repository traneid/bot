ВНИМАНИЕ ДАННАЯ ПРОГРАММА НЕ ГОТОВА !

Описание программы.
Данная программа состоит из трех основных частей.
1. Проверка почты на новое сообщение
2. Обработка и классификация сообщения с помощью библиотеки sklearn и nltk
3. Телеграмм бот для отправки данных (будет отправлять в канал данные для другого бота, номер обращения, классификация обращения, в дальнейшем планируется подставлять исполнителя по схожим задачам)


1 - проверка почты

Алгоритм простой, постоянная проверка почтового сервиса с помощью библиотеки email И smtplib.
Если находит, возвращает текст обращения. (немного в обработанном виде)

2 - Обработка и классификация сообщения
Программа облучается на моем датасете из ексель файла используя метод Pipeline для определения оптимальных данных.
Программа обрабатывает текст, убирая лишнюю информацию и нормализует текст.
Полученный текст программа прогоняет по обученной модели (присваивая токены) и с условными метриками и по возможности классифицирует текст.

3- Телеграмм бот

Бот запускает обучение модели и фоном слушает новые сообщения на почте, если такие есть, сохраняет текст в list.
После команды start программа принимает в себя данные из list вырезая данные из list, после этого отправляет в ii на классификацию обращения и отправляет результат в телеграм, это продолжается пока list не будет пустой (так мы не потеряем поток входящих писем).

Это краткое описание программы.
